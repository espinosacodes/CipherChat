How to Import a Public Key
Step 1: Get the Username

Ask the user for their exact username in CipherChat. This must match exactly.

Step 2: Get the Public Key

Ask the user to export their public key and share it with you. It should look like:

-----BEGIN PUBLIC KEY-----
MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8A...
-----END PUBLIC KEY-----
Step 3: Import

Enter the username and paste the public key data above, then click Import.

Step 4: Verify

The key will be validated and stored. You can then send encrypted messages to that user.

Security Notes
Important: Only import public keys from users you trust and through secure channels.
  • Public keys are safe to share - they cannot be used to decrypt messages
  • Verify the username matches the person you're communicating with
  • Use secure channels (not public forums) to exchange public keys
  • You can always remove imported keys if needed

How Key Exchange Works
Step 1: Initiate

You initiate a key exchange with the selected user. The system will generate a secure exchange request.

Step 2: Notification

The recipient will be notified of your key exchange request and can accept or decline it.

Step 3: Agreement

Once both parties agree, the system will securely exchange public keys and establish encrypted communication.

Step 4: Secure Communication

After successful exchange, you can send encrypted messages to each other using the exchanged keys.
